# Blog-Management-System

📝 Blog Management System

A simple and responsive Blog Management System developed using Python Flask, HTML, and CSS. This web application allows users to create, manage, edit, and delete blog posts through an easy-to-use interface.

📖 Overview

The Blog Management System is a web application designed to simplify blog publishing and management. It enables users to write, edit, update, and delete blog posts while providing a clean and responsive user interface.

This project was developed to gain practical experience with Python Flask, web development concepts, routing, templates, and CRUD (Create, Read, Update, Delete) operations.

✨ Features

User Registration
User Login & Logout
Secure Authentication
Create New Blog Posts
View All Blog Posts
Read Blog Details
Edit Existing Blogs
Delete Blogs
Responsive User Interface
Simple and Easy Navigation

⚙️ Installation

1. Clone the Repository
git clone https://github.com/JaiminJarsaniya/Blog-Management-System.git

3. Move into the Project Folder
cd Blog-Management-System

4. Create a Virtual Environment
python -m venv venv
Windows
venv\Scripts\activate
Linux / macOS
source venv/bin/activate

5. Install Dependencies
pip install -r requirements.txt

6. Run the Application
python app.py

The application will run at:

http://127.0.0.1:5000
